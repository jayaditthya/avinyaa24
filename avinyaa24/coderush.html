About Code Rush


Description :

"CODE RUSH" a coding challenge ,where show off your coding expertise! Take part in our blitz challenge by answering questions about coding, earning valuable seconds to solve the coding challenge, and then solving a blind coding mystery. Prove your speed, wit, and problem-solving ability in this 2-round battle. Think fast, code faster.

Rules :

Solo event.

Consists of 2 rounds.

Strict measures against plagiarism .

Evaluation will be based on your understanding of questions, Time complexity and test case passed

Helping the other participants during the event is not allowed.


Evaluation :

Time complexity of the code.

Time used to solve the code.

Functionality: Does your code generate the correct output for all test cases? This is the ultimate test of your understanding of the problem and the accuracy of your solution.

Readability : Even in the absence of visual cues, can others understand your code? Clear variable names, comments, and proper structure will earn you points.

Error Handling: Evaluates how the code gracefully handles unexpected inputs or errors, ensuring robustness .
